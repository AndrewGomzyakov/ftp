Клиент FTP
Версия: 1.0
Автор: Гомзяков Андрей (avural-k@mail.ru)
Описание:
Данное приложение является реализацией клиента для FTP серверов
Требования:
Python версии не ниже 3.4
Состав
Клиент, обрабатывающий команды: client.py
Класс, реализующий логику: ftp.py
Тесты: tests.py
Способ запуска консольной версии: py -3 client.py -port 21 -adr ftp.mccme.ru

Виды команд клиента:
	CWD dir_name - переход в данную директорию
	PWD - вывод текущей директории
	LIST - содержимое текущей директории
	QUIT - выход
	USER - вход под своим логином
	MKD dir_name - создание директории
	RETR file_name - скачивание файла
	FEAT - особые возможности сервера
	HELP - хелп по серверу
	SIZE file_name - зармер файла
	STOR file_name - загрузка файла
	LA - рекурсивный вывод всех директорий
	LOAD dir_name - рекурсивное скачивание дикеутории
