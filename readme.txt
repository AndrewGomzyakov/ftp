Клиент FTP
Версия: 1.0
Автор: Гомзяков Андрей (avural-k@mail.ru)
Описание:
Данное приложение является реализацией клиента для FTP серверов
Требования:
Python версии не ниже 3.4
Состав
Клиент, обрабатывающий команды: client.py
Класс, реализующий логику: ftp.py
Тесты: tests.py
Способ запуска консольной версии: py -3 client.py -port 21 -adr ftp.mccme.ru
